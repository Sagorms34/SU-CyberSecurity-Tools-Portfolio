## üåü Overview
**SU-AutoReport** is an advanced Python-based penetration testing report generator. It automates basic reconnaissance and security analysis (like port scanning, header compliance checks, and directory enumeration) and consolidates all findings into a professional, industry-standard **HTML Report (Apex Edition)**.

This tool is designed to provide quick and comprehensive security documentation, complete with **CVSS Scores** and **Severity Vectors**.

## üöÄ Features (Apex Edition)
* **Apex HTML Output:** Generates a visually professional, color-coded HTML report.
* **Advanced Scanning:** Checks for open ports, common vulnerable directories, and `robots.txt` path exposure.
* **Security Header Compliance:** Checks for critical HTTP Security Headers (HSTS, CSP, X-Frame-Options, etc.).
* **CVSS Scoring:** Assigns dynamic CVSS Base Scores and Vectors for accurate risk assessment.
* **Clear Recommendations:** Provides specific and actionable remediation advice for each finding.
